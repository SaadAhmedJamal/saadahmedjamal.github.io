4) Spatially Enabled Learning

This was one of the topics I was less familiar with but I found it very interesting. The use of GIS in improving education and cognitive skills. The researcher had students who are going to be future high school teachers. The main research question was how to improve high school education using GIS. Effectively, now Geomedia is being used to foster spatial and analytical thinking in classrooms. The dual coding theory, visual and verbal stimuli and human cognition system sounded interesting. Its been empirically proven that the visual information shows way higher memorability. From activating (learning) point of view both visual and verbal stimuli should be used and the researcher was trying to implement this thing in his slides. Cognitive Load theory was already known that our brain can only perform a limited tasks after which we cannot process anymore but split attention effect was new and interesting. Modern geomedia has potential to minimize split attention effect.

9) Semantic Earth Observation Data Cubes

H Augustin PhD Researcher Introduced Earth Observation Data Cube which reduces complexity by bringing users to the data instead of bringing data to the user and by managing data as multi-dimensional arrays. Sharing the main motivation behind Earth Observation Data Cube told to be Landsat in 2008 making its archive data as public. Now, with increasing data (petabytes), it is becoming more and more important to use data efficiently. Big data is great but there are some challenges, As throughout the world it is a general known fact that data is increasing at a faster rate as compared to computational resources. Also, it is useless to have redundant data at multiple computers. Semantic Data Cube allows data to be stored in a query optimized way instead acquisition optimized way which is more of a logical view on storing data.
